Email Configuration

Email settings are configured via the .env file in the project root.

GMAIL_APP_PASSWORD=xxxx-xxxx-xxxx-xxxx

# Sender identity
SENDER_NAME=Your Name
SENDER_TITLE=Partnership Manager
SENDER_EMAIL=your-email@gmail.com
