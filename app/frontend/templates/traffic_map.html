覆盖率测算方法与改进方向

当前测算方法（Tier 1: 估算）

  • 社媒平台: 覆盖率 = 券商内容占金融内容比例 × 富途在券商内容中份额。例: YouTube 18%(券商/金融) × 50%(富途/券商) ≈ 4%
  • 其他渠道: 基于广告投放观察 + 品牌提及频率 + 官方运营活跃度的综合估算
  • 校准数据: YouTube 频道订阅数对比、Truescope 社交监听、IAB HK 小红书报告等已知锚点

更科学的测算方法（建议升级路径）

Tier 2: 社交监听工具

Meltwater / Truescope / Brandwatch — 按品牌关键词(富途/moomoo/futu)监听各平台提及量, 与竞品(老虎/长桥/IBKR)对比, 计算 Share of Voice

成本: $500-2,000/月 | 精度: 中高

Tier 3: SEM/SEO 工具

SEMrush / Ahrefs — 追踪品牌词+品类词在 Google HK 的搜索量、广告投放份额、自然排名占位, 计算搜索声量份额

成本: $200-500/月 | 精度: 中高(仅限搜索渠道)

Tier 4: 平台付费工具

千瓜/新红(小红书) + 火烧云(B站) — 获取品牌笔记/视频精确数量, 与品类总量对比; 各平台广告后台的 Audience Overlap 数据

成本: ¥500-3,000/月/平台 | 精度: 高

Tier 5: 调研 + API

品牌认知度调研(1,000份HK样本, 问「你在XX平台见过富途内容吗」); 各平台 API 或爬虫获取精确帖子数/曝光量

成本: HK$30,000-80,000/次 | 精度: 最高
